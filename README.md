# Faizan — Personal Portfolio

A responsive, dark-themed personal portfolio built with semantic HTML, modern CSS, and vanilla JavaScript. No frameworks, no build step.

## Features

- Sticky navigation with active-section highlighting
- Hero, About, Services, Projects, and Contact sections
- Scroll-reveal animations (respects `prefers-reduced-motion`)
- Contact form that opens the visitor's email client with a pre-filled message
- Fully responsive down to mobile

## Run locally

It's a static site — just open `index.html`, or serve it:

```bash
python3 -m http.server 8000
# visit http://localhost:8000
```

## Deploy (GitHub Pages)

Repo **Settings → Pages → Build from branch → `main` / root**.
Live at `https://alifaizanblackia-ship-it.github.io/portfolio/`.
